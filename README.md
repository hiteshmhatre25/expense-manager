# Expense Manager

A full-stack expense tracking and personal finance management web application built to help users manage daily expenses, monitor spending habits, and organize financial records efficiently.

---

# Features

* Add and manage expenses
* Track income and spending
* Expense categorization
* Dashboard overview
* Responsive user interface
* REST API integration
* Real-time expense updates
* Data storage using database integration

---

# Installation and Setup

## Clone Repository

```bash
git clone https://github.com/hiteshmhatre25/expense-manager.git
```

---

# Backend Setup

Move to backend folder:

```bash
cd expensemanager
```

Run Spring Boot application:

```bash
./mvnw spring-boot:run
```

For Windows:

```bash
mvnw.cmd spring-boot:run
```

---

# Frontend Setup

Move to frontend folder:

```bash
cd expense-frontend
```

Install dependencies:

```bash
npm install
```

Start frontend:

```bash
npm start
```
